# Onur Çay Evi — QR Menü Sistemi

Geleneksel Türk Çay Evi için profesyonel QR kodlu dijital menü sistemi.

## 🚀 Özellikler

- **Müşteri Menü Sayfası** — Responsive, animasyonlu, mobil uyumlu
- **Admin Paneli** — Ürün, kategori ve ayar yönetimi
- **QR Kod Oluşturucu** — Anında QR kod üretimi ve indirme
- **Fotoğraf Yükleme** — Ürünlere fotoğraf ekleme
- **Arama** — Anlık ürün arama
- **Kategori Filtreleme** — Tab bazlı kategori filtresi

## 🔑 Admin Girişi

- **URL:** `/admin.html`
- **Kullanıcı Adı:** `admin`
- **Şifre:** `onur2024`

> ⚠️ Giriş yaptıktan sonra Ayarlar > Şifre Değiştir bölümünden şifrenizi güncelleyin.

## 🌐 GitHub Pages ile Yayınlama

1. GitHub > Repo > **Settings**
2. Sol menüden **Pages** seçin
3. **Source:** `Deploy from a branch`
4. **Branch:** `main` / `(root)`
5. **Save** — birkaç dakika sonra site yayında!

Menü adresi: `https://[kullaniciadiniz].github.io/onurcayevi/`

## 📱 QR Kod

Admin paneli > **QR Kod** sekmesinden menünüzün QR kodunu oluşturup indirebilirsiniz.
